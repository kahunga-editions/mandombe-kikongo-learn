

# Créer des leçons et exercices à partir du PDF "Exprimer le temps 2"

## Contenu extrait du PDF

Le PDF contient du matériel qui étend la leçon existante "time-expressions" avec :

**Nouveau vocabulaire** (non présent dans la leçon actuelle) :
- Muna mbaji = après-demain
- Mazuji = avant-hier
- Lumbu tshatshi / lumbu ki / lumbu liaki = synonymes d'aujourd'hui
- Mvula za jingi = plusieurs années
- N'mvu mia mingi = plusieurs années (variante)
- Ha manima = à la fin, par la suite
- Mboko = ensuite, après
- Mini mia mbangala = les chaleurs de mbangala
- Nkokela = le soir (déjà dans greetings, pas dans time)
- Mu pari = le matin (contexte)

**Nouveaux verbes et expressions** :
- Vula = défaire | Vula kinkuti = défaire un habit | Vula nsuki = défaire les cheveux
- Vula vula = surtout
- Sa / Shiri = mettre, faire
- Djoka / djokele = courir, s'échapper
- Diata = marcher, écraser
- Pinzumuka = glisser des mains
- Burika / Budika = se casser
- Duka / dukidi = sortir
- Ngantu / nganti = de peur que
- Fueti = devoir (must)
- Kelaka = qui garde

**Phrases complètes (conjugaison avec mazuji/mbaji)** :
- 6 phrases "Mazuji ku [lieu] [pronom] [yele/bele]" (avant-hier + 6 personnes)
- 4 phrases "Mbaji ku [lieu] [pronom] kwenda/kwiza/lomba" (demain + futur)
- Questions temporelles : "Duka wa dukidi mazono mu nkokela?", "Mbo wa ntambila mu pari?"
- Phrases avec vula, fueti, ngantu, etc.

## Plan : 2 nouvelles leçons

### Leçon 1 : "Mazuji ye Mbaji — Avant-hier & Après-demain"
- **id** : `time-advanced`
- **Vocabulaire** (~12 entrées) : mazuji, muna mbaji, lumbu tshatshi, lumbu ki, lumbu liaki, nkokela, mu pari, ha manima, mboko, mini mia mbangala, mvula za jingi, n'mvu mia mingi
- **Phrases** (~12) : les 6 conjugaisons mazuji + les 4 phrases mbaji + les 2 questions temporelles
- **Syntaxe** : un groupe montrant le pattern "Mazuji ku [lieu] [pronom] yele/bele" avec les 6 personnes
- **Exercices** :
  - 3 multiple-choice (mazuji, muna mbaji, questions temporelles)
  - 2 matching (expressions avancées ↔ traductions)
  - 2 fill-in-blank (compléter les phrases)
  - 1 mandombe-recognition (8 items : mazuji, muna mbaji, nkokela, mu pari, ha manima, mboko, lumbu tshatshi, mvula za jingi)

### Leçon 2 : "Vula, Djoka, Diata — Verbes d'action"
- **id** : `action-verbs-vula`
- **Vocabulaire** (~14 entrées) : vula, vula kinkuti, vula nsuki, vula vula, sa/shiri, djoka/djokele, diata, pinzumuka, burika/budika, duka/dukidi, ngantu/nganti, fueti, kelaka, mpe
- **Phrases** (~8) : nsuki ni ta vula, nsuki zani mvuridi, nge fueti zaba ti, ambe tshima tsho tsha mambu, fueti kue tsha kele, ngati tshi djokele, ni ka shiri, we keti St Pierre kelaka muelo zulu
- **Syntaxe** : groupe montrant les formes de "vula" (présent continu, passé composé)
- **Exercices** :
  - 3 multiple-choice
  - 2 matching
  - 1 fill-in-blank
  - 1 mandombe-recognition (8 items : vula, djoka, diata, pinzumuka, burika, duka, fueti, ngantu)
