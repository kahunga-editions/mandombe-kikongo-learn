

# Nouvelles leçons à partir de "Kikongo 2" — Animaux, Corps & Concepts

## Contenu du PDF (50 pages)

Le PDF est un dictionnaire illustré organisé par préfixe syllabique (bi-, bu-, be-, ng-, nk-). Après déduplication avec les leçons existantes, voici le contenu nouveau :

**Animaux (14 mots)** : ngamalu (chameau), ngombe (vache), ngoko (taureau), ngongolo (chenille), ngola (poisson-chat), nioka (serpent), nkala (crabe), nkami (fourmi rouge géante), nkamvu (bélier), nkelele (pintade), nkusu (perroquet), nsombe (ver de palmier), buluku (âne), ngulu ya mfinda (phacochère)

**Corps & Actions physiques (12 mots)** : bunda (cuisse/hanche), bundi (mâchoire/joue), beni/mabeni (sein/seins), bito (pattes d'animal), bueta (écraser), buila (attraper), buka (guérir), benda (traîner/se dépêcher), benga (gouffre), beta (aplatir/gronder), betama (se pencher/s'agenouiller), bumba (étreindre)

**Vie sociale & Concepts (12 mots)** : bumwana (enfance), bukuluntu (aînesse), buyaya (qualité d'être plus âgé), buzitu (respect), zitisa (respecter), bukundi (faveur/grâce), buatu (pirogue), bulu (trou), binsono (écrits), bungu (motte de terre), bimba (goûter), bioka (roter)

**Nourriture & Médecine** : bilongo (médecine), bidia (nourriture), bima/kima/tshima (choses/outils), ndala (feuille de palmier)

**Illustrations** : Le PDF contient des illustrations colorées d'animaux (âne, taureau, serpent, crabe, perroquet, pintade) avec les glyphes Mandombe. Ces images seront copiées dans le projet pour enrichir visuellement les leçons.

## Plan : 3 nouvelles leçons + illustrations

### Leçon 1 : `animals-bibulu` — "Bibulu ye Binyama — Animaux"
- **Vocabulaire** (14) : tous les animaux listés ci-dessus
- **Phrases** (6) : phrases contextuelles avec les animaux
- **Syntaxe** : bloc montrant le pattern singulier/pluriel pour les noms d'animaux
- **Exercices** :
  - 3 multiple-choice (identifier l'animal, traduire)
  - 2 matching (animal ↔ traduction)
  - 2 fill-in-blank
  - 1 mandombe-recognition progressif (10 items) :
    - Items 1-4 (debutant) : mots courts `glyph-to-latin` (nkala, nioka, ngombe, nkusu)
    - Items 5-7 (intermediaire) : mots plus longs `glyph-to-latin` (ngamalu, nkelele, ngongolo)
    - Items 8-10 (avance) : `latin-to-glyph` (buluku, nkamvu, nsombe)

### Leçon 2 : `body-actions` — "Nitu ye Bisalu — Corps et Actions"
- **Vocabulaire** (12) : bunda, bundi, beni, mabeni, bito, bueta, buila, buka, benda, beta, betama, bumba
- **Phrases** (8) : phrases d'usage avec les verbes d'action
- **Syntaxe** : bloc montrant l'impératif des verbes d'action (buka, benda, buila)
- **Exercices** :
  - 3 multiple-choice
  - 2 matching (verbe ↔ action)
  - 1 fill-in-blank
  - 1 mandombe-recognition progressif (10 items) :
    - Items 1-4 (debutant) : `glyph-to-latin` (buka, benda, beta, beni)
    - Items 5-7 (intermediaire) : `glyph-to-latin` (bueta, bumba, betama)
    - Items 8-10 (avance) : `latin-to-glyph` (bunda, buila, bundi)

### Leçon 3 : `social-concepts` — "Buzitu ye Bumwana — Respect et Enfance"
- **Vocabulaire** (14) : bumwana, bukuluntu, buyaya, buzitu, zitisa, bukundi, buatu, bulu, binsono, bungu, bimba, bioka, bilongo, ndala
- **Phrases** (8) : phrases avec les concepts abstraits
- **Syntaxe** : bloc montrant les noms abstraits en bu- (bu + racine = concept abstrait)
- **Exercices** :
  - 3 multiple-choice
  - 2 matching
  - 1 fill-in-blank
  - 1 mandombe-recognition progressif (10 items) :
    - Items 1-4 (debutant) : `glyph-to-latin` (buzitu, bulu, buatu, ndala)
    - Items 5-7 (intermediaire) : `glyph-to-latin` (bilongo, binsono, bumwana)
    - Items 8-10 (avance) : `latin-to-glyph` (bukuluntu, bukundi, buyaya)
