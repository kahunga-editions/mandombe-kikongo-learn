## Plan : 3 nouvelles lecons

### Lecon 1 — "Bisakumunu na Mutoto" (Salutations na la Terre)
- **Niveau** : beginner
- **Vocabulaire** (~15 mots) : mutoto, mutoto wa mukaka, mutoto wa mvimba, mbote zêno, mbote aku, mbote zawu, batata mbote zawu, lu kolele, vumbukidi, lu vumbukidi, sikama, shama, chemi, lu chemi, sala
- **Syntaxe** : bloc sur les formes de salutation (singulier aku / pluriel zêno / 3e pers. zawu), bloc sur les expressions de bien-etre (kolele/vumbukidi/chemi)
- **Exercices** :
  - Multiple-choice (salutations)
  - Matching (expression Lari ↔ traduction)
  - Fill-in-blank (completer une salutation)
  - Mandombe-recognition progressif (8 items : 4 glyph-to-latin debutant, 2 glyph-to-latin intermediaire, 2 latin-to-glyph avance)

### Lecon 2 — "Bifua bia binsiku — mu/ba, mu/mi" (Classes nominales, partie 1)
- **Niveau** : intermediate
- **Vocabulaire** (~20 mots) : les paires singulier/pluriel mu/ba (muntu/bantu, muana/bala, mutekolo/batekolo, nkaka/bankaka, nkento/bakento), mu/mi (mulele/milele, muti/miti, mulembo/milembo, mulumba/milumba, mutshila/mitshila), N-invariable (nsonso, mbulu, mbuma, mbua), mbutukulu
- **Syntaxe** : bloc explicatif sur le systeme de prefixes mu→ba (humains), mu→mi (objets/nature), N-invariable (animaux)
- **Exercices** :
  - Multiple-choice (identifier le pluriel)
  - Matching (singulier ↔ pluriel)
  - Fill-in-blank (prefixe manquant)
  - Mandombe-recognition progressif (8 items : debutant mots courts, intermediaire paires, avance latin-to-glyph)

### Lecon 3 — "Bifua bia binsiku — ki/bi, lu/ma, ku/ma" (Classes nominales, partie 2)
- **Niveau** : intermediate
- **Vocabulaire** (~24 mots) : ki/bi (kizengi/bizengi, kikoba/bikoba, kibanga/bibanga, kiamvu/biamvu, kihota/bihota, kizongo/bizongo), le/ma (lembe/malembe), lu/ma (lukaya/makaya), lu/bu (luwa/buwa), lu/tu (lumbembemba/tumbembemba, lubo/tubo), lu/mbue (lumbuetete/mbuetete), ku/ma (kulu/malu, kutu/makutu, kuku/makuku), zu/ma (zu/mazu), la/ma (lala dia nsa/malala dia nsa), bo/ma (boria/maboria)
- **Syntaxe** : bloc par groupe de prefixes avec exemples
- **Exercices** :
  - Multiple-choice (quel prefixe pour le pluriel)
  - Matching (singulier ↔ pluriel)
  - Fill-in-blank
  - Mandombe-recognition progressif (8 items avec difficulte graduelle)
