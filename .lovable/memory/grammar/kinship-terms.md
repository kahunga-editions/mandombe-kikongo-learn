---
Le systeme de parente en Lari (corpus Nzo Mikanda) :

- tata / ta = pere
- mama / ma = mere
- nkaji = epouse
- mwana = enfant
- yaya / ya = grand-parent
- mbutu = oncle maternel

Regles :
- JAMAIS de doubles lettres : tata (pas taata), mama (pas maama), yaya (pas giaagia)
- "mwana" est autorise comme vocabulaire mais JAMAIS comme terme d'adresse pour les eleves
- Utiliser "nlongoki" (eleve) pour s'adresser aux apprenants
