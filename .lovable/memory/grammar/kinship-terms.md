name: Kinship Terms
description: Lari kinship terms with Nzo Mikanda orthography (no double letters)

---
Le systeme de parente en Lari (corpus Nzo Mikanda) :

- tata / ta = pere
- mama / ma = mere
- nkaji = epouse
- mwana = enfant
- yaya / ya = aine, grand-parent (JAMAIS la graphie "giagia" — utiliser "yaya")
- mbutu = oncle maternel
- Appellatifs avec nom propre : "ta Malonga" = Pere/Monsieur Malonga ; "ma Malonga" = Oncle Malonga (Malonga est un nom d'homme, "ma" devant un nom masculin = appellatif respectueux pour un oncle) ; "ya Malonga" = Aine/Grand-Parent Malonga.

Regles :
- JAMAIS la graphie "giagia" / "giaagia" — toujours "yaya" (pluriel "mayaya")
- JAMAIS de doubles lettres erronees : tata, mama, yaya
- JAMAIS la structure Kituba "beto + verbe" pour "nous + verbe" : utiliser le prefixe verbal "tu-" (ex. "tu kwenda" = nous allons/irons)
- "mwana" est autorise comme vocabulaire mais JAMAIS comme terme d'adresse pour les eleves
- Utiliser "nlongoki" (eleve) pour s'adresser aux apprenants
